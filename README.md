# Skylaws Official Website

A modern, responsive artist portfolio website built for **Skylaws**, showcasing music, creativity, and visual expression.

---

## 🌐 Live Preview

https://paakwesidev.netlify.app

---

## 🎯 Project Overview

This website is a clean and minimal personal brand site designed for an artist.  
It focuses on:

- Professional online presence
- Music/video showcasing
- Social media connection
- Easy contact access

---

## ✨ Features

- 🎬 Modern hero section with smooth intro transition
- 🎨 Minimal dark-themed premium design
- 📱 Fully responsive (mobile + desktop)
- 📺 YouTube embedded featured content
- 🔗 Social media integration (Instagram & Snapchat)
- 📩 Clickable email contact link
- ⚡ Lightweight and fast-loading

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

No frameworks or external dependencies used.
